Micro Learning Game

부모가 금방 이해하고 바로 써먹는 ‘작은 신호’

상황별 배경 × 나이대 표정 아트 조합. 라운드마다 짧은 피드백과 부모용 메시지를 제공해요.

데모 실행 제휴/이벤트 문의

코어 루프

  • ① 상황 제시 → ② 아이 표정/행동 선택 → ③ 올바른 부모 메시지 확인
  • 짧은 라운드(60–90초)로 반복

확장

  • 나이대(36·42·48·60개월) 아트팩
  • 상황팩(외출/양치/식사/수면/감정)
  • 앱인토스/스토어 이벤트 연동
